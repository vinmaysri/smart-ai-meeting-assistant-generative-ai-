# 🤖 Smart AI Meeting Assistant

An AI-powered web application that converts unstructured meeting transcripts into **professional, structured meeting minutes** using Generative AI.

Built using **Python (Flask)** and **Google Generative AI (Gemini API)** with a clean and modern UI.

---

## 🚀 Features

- 🧠 AI-based meeting minutes generation  
- 📋 Structured output including:
  - Summary  
  - Key Discussion Points  
  - Decisions Made  
  - Action Items  
- 📊 Action items displayed in table format  
- 📋 Copy to clipboard functionality  
- ⬇ Download meeting minutes as a text file  
- 🎨 Modern and responsive UI  
- ⚡ Fast and interactive interface  

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** Python (Flask)  
- **AI Model:** Google Generative AI (Gemini API)

---

## 🏗 Project Architecture

User Input (Meeting Transcript)

↓
Frontend (HTML/CSS - Web Interface)

↓
Flask Backend (Python)

↓
Prompt Engineering Layer

↓
Generative AI Model (Gemini API)

↓
Structured Meeting Minutes (HTML Output)

↓
Display on UI + Copy/Download Feature
